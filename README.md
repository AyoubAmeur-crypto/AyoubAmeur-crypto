## About

I build things that think. My work sits at the crossroads of **full-stack engineering**, **AI/ML systems**, and **mobile development** — from crafting scalable APIs to training models that actually ship. I care about clean architecture, intuitive interfaces, and writing code that reads as well as it runs.

When I'm not deep in a codebase, I'm behind a camera. I work part-time as a videographer and photographer — a discipline that has made me a sharper designer and a more deliberate thinker. I love to travel; every new place rewires how I see problems.
